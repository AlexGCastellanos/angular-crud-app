# angular-crud-app

En esta app podrás crear, editar, consultar y eliminar clientes, 
este repositorio GitHub corresponde al Frontend hecho en Angular que conecta con
la API Rest creada con Spring Framework que conecta al repositorio de clientes
de una de base de datos MySQL. 

-Si quieres probar el frontend ve aquí --> https://angular-clientes-app.herokuapp.com/

-Si quieres ver el repositorio del API Rest en el Backend ve aquí --> https://github.com/AlexGCastellanos/springboot-backend-apirest

## Tecnologías usadas

* 📐Angular
* 🍃Spring Boot
* ☕Java SE
* 🧾Spring Data
* 💾Hibernate
* 💼Jakarta EE
* 🟧HTML
* 🟪Bootstrap
